# Weather App

A simple Weather Application built using React, TypeScript, Vite, and Zustand.

## Features

- Search weather by city name
- Display temperature, humidity, and wind speed
- Loading state handling
- Error handling for invalid cities
- Global state management using Zustand
- TypeScript support

## Technologies Used

- React
- TypeScript
- Vite
- Zustand
- Axios
- OpenWeather API
